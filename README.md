# Adidas EE. UU. Sales Dashboard

Repositorio con el dashboard interactivo de Tableau que analiza el desempeño de ventas de Adidas en Estados Unidos. El panel facilita la exploración de tendencias mensuales, rendimiento por estado y retailer, y la distribución por canal de venta.

---

## Objetivo del proyecto

Proveer una vista ejecutiva para responder preguntas clave de negocio:
- Evolución mensual de ventas y estacionalidad
- Estados y regiones con mayor aporte de ventas
- Retailers que generan más ingresos
- Distribución por canal (In-store, Outlet, Online)

El dashboard incluye filtros interactivos por año, región y categoría de producto para profundizar en segmentos concretos.

---

## Principales hallazgos 

- Patrones de estacionalidad con picos relevantes.
- Contribución desbalanceada por estados y pocos retailers dominantes.
- Canales en tienda (In-store) y outlet concentran gran parte del volumen.
